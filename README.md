# C-DecisionLogic-Lab

## Overview

C-DecisionLogic-Lab is a collection of beginner-to-intermediate C programming exercises focused on conditional statements, switch-case constructs, mathematical computations, and real-world problem-solving.

The repository is designed to strengthen logical thinking and programming fundamentals by implementing practical applications using core C language concepts.

---

## Concepts Covered

- Decision Making using if, else if, and else
- Nested Conditional Logic
- Switch-Case Statements
- Mathematical Computations
- Standard Math Library Functions
- Real-world Programming Applications
- Input and Output Operations
- Problem Solving using C
